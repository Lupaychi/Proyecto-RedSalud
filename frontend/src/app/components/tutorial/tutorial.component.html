Guía rápida de uso

¡Bienvenido! Aquí aprenderás a utilizar cada sección de la aplicación RedSalud.
  1. Asignaciones
    • Haz clic en Nueva asignación para asignar un médico a un box disponible.
    • Consulta todas las asignaciones existentes en Ver asignaciones.
    • Tip: Puedes editar o eliminar asignaciones desde la lista.
  2. Boxes
    • Agrega un nuevo box usando Crear box, indicando piso y detalles relevantes.
    • Gestiona la disponibilidad de boxes para una mejor organización.
  3. Sistema
    • Verifica el estado del sistema y la conexión en Estado del sistema.
    • Si ves algún error, contacta al administrador.
  4. Directorio Médico
    • Consulta todos los médicos y sus especialidades en Ver directorio.
    • Utiliza la búsqueda para encontrar médicos rápidamente.
  5. Reportes y Tutorial
    • En esta sección puedes acceder a este tutorial para aprender a usar la app.
    • Próximamente podrás ver estadísticas y reportes de ocupación.
¿Dudas? Revisa cada sección desde el Panel de Control o contacta al administrador para soporte.
